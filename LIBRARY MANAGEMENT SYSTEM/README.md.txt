‚≠ê INTRODUCTION ‚≠ê

Welcome to the Library Management System!

This is a simple yet effective program developed in C that allows you to manage books in a library. The system provides functionalities to:

Add new books to the library.

List all books.

Search for a book by its ID.

Borrow a book.

Return a borrowed book.

The system efficiently stores book details such as ID, title, author, borrowing status, and student ID (if borrowed).

‚úíÔ∏è FEATURES ‚úíÔ∏è

1. Add Book:

Allows you to add a new book to the library by entering:

Book ID

Title

Author

2. List Books:

Displays all books in the library along with their details:

ID

Title

Author

Borrowing status (Available or Borrowed)

3. Search Book:

Allows you to search for a book by its ID and display its details.

4. Borrow Book:

Enables a student to borrow a book by entering:

Book ID

Student ID

If the book is already borrowed, a message will be displayed.

5. Return Book:

Allows a student to return a borrowed book by entering the book ID.

If the book is already available, an appropriate message will be shown.

6. Exit:

Safely exits the program.

üîß HOW TO USE üîß

Step-by-Step Guide:

Run the Program:Upon execution, the program will display a menu with different options.

Choose an Operation:Select an operation from the menu:

1: Add a book

2: List all books

3: Search for a book

4: Borrow a book

5: Return a book

6: Exit

Enter Required Information:Depending on the operation selected, you will be prompted to enter book details (ID, title, author) or student details (for borrowing).

View Book List:Selecting "List all books" will display all books along with their details and status.

Exit the Program:Select "Exit" to safely close the program.
